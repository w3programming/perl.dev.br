Setting Up the Perl Development Environment

In this tutorial we are going to install Perl and everything else necessary to start programming in this language.

To program in Perl it’s necessary to install the programming language distribution application and a simple text editor.

Strawberry Perl — What is it and how to download it

Strawberry Perl is a Perl development environment for Microsoft Windows.

Strawberry Perl contains all the requirements needed to run Perl scripts.

Strawberry Perl can be downloaded directly from the official website https://strawberryperl.com  .

After downloading it, just run the installer application.

Installing Strawberry Perl on Windows — Step by Step with Images

  1. Right after running the Strawberry Perl installer, the window below appears. Click the NEXT button to continue.

  1. Read and accept the program terms by clicking the I accept the terms in the license agreement checkbox and then the next button

  1. Select the application installation location. By default, it is suggested to install Strawberry Perl in the C:\Strawberry folder. If in doubt where to install it, leave this default folder.

  1. With the whole system configured, just click the Install button to start the software installation.

  1. Now, just wait for the installation to complete.

  1. After completing the Strawberry Perl installation, the following screen appears. Finish the installer by clicking the finish button.

Perl installation confirmation

To confirm that Perl has been properly installed on your computer, follow these steps:

  1. Open a Windows command prompt and type the command perl -v.
  2. Press Enter to execute the command.
  3. Check if the command outputs the Perl version installed on your computer as shown in the image below.

If the command returns something like this:

'perl' is not recognized as an internal or external command, operable program or batch file.

or

'perl' is not recognized as an internal command...

…it is necessary to verify if the directory C:\Strawberry\perl\bin has been added to the windows PATH environment variable.

You can add the Perl directory to the PATH environment variable using the command:

setx path "%path%;C:\Strawberry\perl\bin"

Installing a Text Editor — Also Known as a Code Editor

Perl scripts are plain text.

Perl scripts can be edited in any plain text editor.

To make programming in Perl easier, it’s recommended to install an editor that has features such as syntax highlighting and autocompletion.

The editor options are many. I recommend using Visual Studio Code.

To download it, just visit the official website at https://code.visualstudio.com  .

Installing Visual Studio Code is quite simple and straightforward. Run the installer and follow the steps presented by the program.

If you want to try other editor options, you can try Sublime Text  or the Atom  . Two of the most popular code editors among programmers. Notepad++  is also a good option.
