Developing the First Program in Perl

Perl scripts are plain text.

Despite its simplicity, Perl scripts can perform complex tasks.

  • First, create a folder called perl-scripts in the C: root directory.
  • Open your favorite text editor to create the first Perl script.
  • Create a new file and enter the code below:
#!/usr/bin/perl
use warnings;
print("Hello, World!\n");
  • Save the file with the name HELLO.PL to the perl-scripts folder.

Running a script in Perl

To run a Perl script, open a command prompt in Windows.

Navigate to the folder where the script is located.

In our case the folder is located at c:\perl-scripts. To place the prompt inside this folder, type the command cd c:\perl-scripts and press the ENTER key.

Run the command:

perl hello.pl

Output:

Hello, World!

If you see this result on the screen, congratulations you have just created and successfully run your first Perl script.

Explaining the script — In detail

Let’s go through each line of our first Perl program to understand how it works.

#!/usr/bin/perl

The first line tells the shell that this script should be run using the Perl interpreter.

The address /usr/bin/perl is exactly where the interpreter is located.

use warnings;

The second line of the script includes a Pragma.

This Pragma tells the Perl interpreter that additional warnings should be displayed to the user when something happens in the code.

Another very important Pragma used in Perl is use stricts. We’ll see more about it as the tutorial progresses.

print("Hello, World!\n");

The third line of the script contains a function called print.

This function places a value on the screen.

In this case, the value is whatever is inside the parentheses.

"Hello, World!\n" represents a string in Perl.

Strings in Perl are enclosed in quotes "".

Throughout the tutorial, you will learn more about strings in Perl.

The sequence \n within the string represents a newline for the Perl interpreter.

The \n is known as the escape sequence.
