Introduction to Perl

In this article you will learn about some important aspects of the Perl programming language. This is a first contact with characteristics that make up the language.

Perl Features — Important Aspects of the Language

Perl is one of the most popular and feature-rich programming languages ​​on the market. It has been developed and improved for over two decades. It has an active and participatory community in its development.

Perl can run on an insane amount of platform. Compatible with most popular operating systems like Windows, Linux and MacOS, it becomes a wise choice for anyone who needs cross-platform portability.

Perl currently supports over 100 different platforms.

But what can Perl be used for?

Perl is a versatile language. It can be used in both mission-critical and smaller projects, or even very simple one-line scripts.

Perl’s Most Important Features — Language Highlights

1. Extensible programming language

Perl has a module repository called CPAN  , short for Comprehensive Perl Archive Network , in it you can find hundreds of thousands of extensions for the language.

If you need a module that performs a certain task, chances are, someone in the community has already developed it.

Modules range from graphical user interface UIs to integration with the most popular database managers .

Perl provides support for the most popular databases for programmers like MySQL, Oracle, PostgresSQL and more.

2. Ease of use and learning

Despite some controversies regarding Perl, the language is very easy to learn.

In addition to having a huge and helpful community  , you can still find thousands of tutorials available for free on the internet.

Perl also has documentation  very detailed and easy to understand. The language stands out for its efficiency in performing complex tasks.

3. Versatility

One of the most important features of any programming language is its versatility regarding programming paradigms.

Object-oriented, procedural programming, or functional programming , all of these programming paradigms are supported by Perl.

This allows the programmer the freedom to develop their programs using the paradigm that best fits their projects.

4. Regular expression

When we talk about text manipulation, Perl is on another level. We can say that the language is the precursor of regular expressions.

Regular expression or regex is a powerful system used to find text occurrences in various types of content. You will learn much more about this system as the tutorial progresses.

Perl has a set of built-in APIs used specifically for text processing. With them it is much easier to manipulate files generated from markup languages ​​such as XML and HTML.

Perl’s regex engine needs no comments. With a robust system, Perl’s regex can be used to find specific files on systems through complex combinations.

5. Open source

An open source project allows the user to access, change and even distribute it freely and freely.

Perl has been developed since its inception as an open source project. You can download Perl, change it, and freely distribute it using one of the GNU, GPL, or artistic licenses.

The language still has support for commercial use.

6. Quality and safety

Language security is essential when we talk about programming.

Perl is certified by Coverity free of security flaws. The language core also has low defect density.

This allows the user to have more confidence in developing their programs using Perl.

Different Perl Versions — Version 5 x Version 6

It is worth noting here that Perl has two major versions that generate confusion among users.

Until version 5 of Perl, the language is still called Perl. Version 6, however, differed so much from Perl that it became another language, known as Raku  .

Raku can be considered a fully object-oriented implementation of Perl.

In our tutorial we are going to use Perl version 5.3.

Who invented Perl?

Perl was first introduced in 1987 by programmer Larry Wall.

He was a systems administrator at NASA and needed a language to process reports more efficiently. The first version of Perl was created and run on the UNIX operating system.

For more than two decades Perl has been developed and improved by an active and dedicated community of programmers.
