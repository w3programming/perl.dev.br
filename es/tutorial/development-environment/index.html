Configuración del Entorno de Desarrollo de Perl

En este tutorial vamos a instalar Perl y todo lo necesario para empezar a programar en este lenguaje.

Para programar en Perl es necesario instalar la aplicación de distribución del lenguaje de y un editor de texto sencillo.

Strawberry Perl — Qué es y cómo descargarlo

Strawberry Perl es un entorno de desarrollo Perl para Microsoft Windows.

Strawberry Perl contiene todos los requisitos necesarios para ejecutar scripts de Perl.

Strawberry Perl se puede descargar directamente desde el sitio web oficial https://strawberryperl.com  .

Después de descargarlo, simplemente ejecute la aplicación de instalación.

Instalación de Strawberry Perl en Windows: paso a paso con imágenes

  1. Inmediatamente después de ejecutar el instalador de Strawberry Perl, aparece la siguiente ventana. Haga click en el siguiente botón para continuar.

  1. Lea y acepte los términos del programa haciendo clic en la casilla I accept the terms in the license agreement y luego en el botón Next

  1. Seleccione la ubicación de instalación de la aplicación. De manera predeterminada, se sugiere instalar Strawberry Perl en la carpeta C:\Strawberry. Si tiene dudas sobre dónde instalarlo, deje esta carpeta predeterminada.

  1. Con todo el sistema configurado, simplemente haga clic en el botón Instalar para iniciar la instalación del software.

  1. Ahora, solo espera a que se complete la instalación.

  1. Después de completar la instalación de Strawberry Perl, aparece la siguiente pantalla. Finalice el instalador haciendo clic en el botón Finish.

Confirmación de instalación de Perl

Para confirmar que Perl se ha instalado correctamente en su computadora, siga estos pasos:

  1. Abra un símbolo del sistema de Windows y escriba el comando perl -v.
  2. Presione la tecla Enter para ejecutar el comando.
  3. Verifique si el comando genera la versión de Perl instalada en la computadora como se muestra en la imagen a continuación.

Si el comando devuelve algo como esto:

'perl' no se reconoce como un comando interno o externo, un programa ejecutable o un archivo por lotes.

o

'perl' não é reconocido como um comando interno...

…es necesario verificar si el directorio C:\Strawberry\perl\bin se ha agregado a la variable de entorno PATH de Windows.

Puede agregar el directorio Perl en la variable de entorno PATH usando el comando:

setx path "%path%;C:\Strawberry\perl\bin"

Instalación de un editor de texto, también conocido como editor de código

Los scripts de Perl son texto sin formato.

Los scripts de Perl se pueden editar en cualquier editor de texto sin formato.

Para facilitar la programación en Perl, se recomienda instalar un editor que tenga funciones como resaltado de sintaxis y autocompletado.

Las opciones del editor son muchas. Recomiendo usar Visual Studio Code.

Para descargarlo, simplemente visite el sitio web oficial en https://code.visualstudio.com  .

La instalación de Visual Studio Code es bastante simple y directa. Ejecute el instalador y siga los pasos presentados por el programa.

Si quieres probar otras opciones del editor, puedes probar Sublime Text  o el Atom  . Dos de los editores de código más populares entre los programadores. Notepad++  también es una buena opción.
