Desarrollo del Primer Programa en Perl

Los scripts de Perl son texto sin formato.

A pesar de su simplicidad, los scripts de Perl pueden realizar tareas complejas.

  • Primero, cree una carpeta llamada perl-scripts en el directorio raíz C:.
  • Abra su editor de texto favorito para crear el primer script de Perl.
  • Cree un nuevo archivo e ingrese el código a continuación:
#!/usr/bin/perl
use warnings;
print("Hello, World!\n");
  • Guarde el archivo con el nombre hello.pl en la carpeta perl-scripts.

Ejecutar un script en Perl

Para ejecutar un script de Perl, abra un símbolo del sistema en Windows.

Navegue a la carpeta donde se encuentra el script.

En nuestro caso, la carpeta se encuentra en c:\perl-scripts. Para colocar el indicador dentro de esta carpeta, escriba el comando cd c:\perl-scripts y presione la tecla Enter.

Ejecute el comando:

perl hello.pl

Salida:

Hello, World!

Si ve este resultado en la pantalla, felicitaciones, acaba de crear y ejecutar con éxito su primer script Perl.

Explicando el guión — En detalle

Repasemos cada línea de nuestro primer programa Perl para entender cómo funciona.

#!/usr/bin/perl

La primera línea le dice al shell que este script debe ejecutarse usando el intérprete de Perl.

La dirección /usr/bin/perl es exactamente donde se encuentra el intérprete.

use warnings;

La segunda línea del script incluye un Pragma.

Este Pragma le dice al intérprete de Perl que se deben mostrar advertencias adicionales al usuario cuando algo sucede en el código.

Otro Pragma muy importante utilizado en Perl es use stricts. Veremos más sobre esto a medida que avance el tutorial.

print("Hello, World!\n");

La tercera línea del script contiene una función llamada print.

Esta función coloca un valor en la pantalla.

En este caso, el valor es lo que está dentro de los paréntesis.

"Hello, World!\n" representa una string en Perl.

Las strings en Perl están entre comillas "".

A lo largo del tutorial, aprenderá más sobre strings en Perl.

La secuencia \n dentro de la string representa una nueva línea para el intérprete de Perl.

El \n es conocido como el escape sequence.
