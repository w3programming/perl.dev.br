Разработка первой программы на Perl

Perl-скрипты представляют собой обычный текст.

Несмотря на свою простоту, Perl-скрипты могут выполнять сложные задачи.

  • Сначала создайте папку с именем perl-scripts в корневом каталоге C:.
  • Откройте свой любимый текстовый редактор, чтобы создать первый Perl-скрипт.
  • Создайте новый файл и введите код ниже:
#!/usr/bin/perl
use warnings;
print("Hello, World!\n");
  • Сохраните файл с именем hello.pl в папку perl-scripts.

Запуск скрипта на Perl

Чтобы запустить Perl-скрипт, откройте командную строку в Windows.

Перейдите в папку, в которой находится скрипт.

В нашем случае папка находится в c:\perl-scripts. Чтобы поместить приглашение в эту папку, введите команду cd c:\perl-scripts и нажмите клавишу Enter.

Запустите команду:

perl hello.pl

Вывод:

Hello, World!

Если вы видите этот результат на экране, поздравляем, вы только что создали и успешно запустили свой первый Perl-скрипт.

Объяснение сценария — Подробно

Давайте пройдемся по каждой строке нашей первой программы на Perl, чтобы понять, как она работает.

#!/usr/bin/perl

Первая строка сообщает оболочке, что этот сценарий должен быть запущен с использованием интерпретатора Perl.

Адрес /usr/bin/perl находится именно там, где находится интерпретатор.

use warnings;

Вторая строка скрипта включает файл Pragma.

Этот Pragma сообщает интерпретатору Perl, что пользователю должны отображаться дополнительные предупреждения, когда что-то происходит в коде.

Другой очень важный Pragma, используемый в Perl, — это use stricts. Мы увидим больше об этом по ходу обучения.

print("Hello, World!\n");

Третья строка скрипта содержит функцию print.

Эта функция выводит значение на экран.

В этом случае значением является то, что находится внутри круглых скобок.

"hello, world!\n" представляет собой строку в Perl.

Строки в Perl заключаются в кавычки "".

В этом руководстве вы узнаете больше о строках в Perl.

Последовательность \n в строке представляет новую строку для интерпретатора Perl.

\n известен как escape sequence.
