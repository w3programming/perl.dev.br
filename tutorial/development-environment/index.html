Configurando o Ambiente de Desenvolvimento do Perl

Nesse tutorial nós vamos instalar o Perl e tudo o que for necessário para começar a programar nesta linguagem.

Para programar em Perl é necessário instalar o aplicativo de distribuição da linguagem de programação e um editor de texto simples.

Strawberry Perl — O que é e como baixá-lo

O Strawberry Perl é um ambiente de desenvolvimento Perl para o Microsoft Windows.

O Strawberry Perl contém todos os requisitos necessários para executar scripts em Perl.

O Strawberry Perl pode ser baixado diretamente do site oficial https://strawberryperl.com  .

Após o download, basta executar o aplicativo de instalação.

Instalando o Strawberry Perl no Windows — Passo a Passo com imagens

  1. Logo após executar o instalador do Strawberry Perl, a janela abaixo é apresentada. Clique no botão NEXT para continuar.

  1. Leia e aceite os termos do programa clicando na caixa de seleção I accept the terms in the license agreement e em seguida no botão Next.

  1. Selecione o local de instalação do aplicativo. Por padrão, é sugerido instalar o Strawberry Perl na pasta C:\Strawberry. Se tiver em dúvida de onde instalá-lo, deixa essa pasta padrão mesmo.

  1. Com o sistema todo configurado, basta clicar no botão Install para iniciar a instalação do software.

  1. Agora, basta aguardar a finalização da instalação.

  1. Após completar a instalação do Strawberry Perl, a seguinte tela é apresentada. Finalize o instalador clicando no botão Finish.

Confirmação da instalação do Perl

Para confirmar se o Perl foi devidamente instalado no seu computador, siga os seguintes passos:

  1. Abra um prompt de comando do Windows e digite o comando perl -v.
  2. Pressiona a tecla Enter para executar o comando.
  3. Verifique se a saída do comando indica a versão do Perl instalado no computador como mostrado na imagem abaixo.

Caso retorno do comando seja algo do tipo:

'perl' is not recognized as an internal or external command, operable program or batch file.

ou

'perl' não é reconhecido como um comando interno...

…é necessário verificar se o diretório C:\Strawberry\perl\bin foi adicionado na variável de ambiente PATH do windows.

Você pode adicionar o diretório do Perl na variável de ambiente PATH usando o comando:

setx path "%path%;C:\Strawberry\perl\bin"

Instalando um editor de texto — Também conhecido como editor de código

Os scripts em Perl são textos simples.

Os scripts em Perl podem ser editados em qualquer editor de texto simples.

Para facilitar a programação em Perl, é recomendável instalar um editor que possua recursos como detaque de sintaxe e autocompletar.

As opções de editores são várias. Eu recomendo usar o Visual Studio Code.

Para baixá-lo, basta visitar o site oficial em https://code.visualstudio.com  .

A instalação do Visual Studio Code é bem simples e direta. Rode o instalador e siga as etapas apresentadas pelo programa.

Caso queira experimentar outras opções de editores, você pode testar o Sublime Text  ou o Atom  . Dois dos mais populares editores de código entre programadores. O Notepad++  também é uma boa opção.
