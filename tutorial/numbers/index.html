Perl - Números

Número inteiro em Perl

Em Perl, é possível trabalhar com números em formato inteiro.

Um número inteiro em Perl não possui dígitos decimais.

Um número inteiro em Perl é um decimal com base 10.

Exemplo:

$x = 10;
$y = 1200;
$z = -2;

Números inteiros em Perl podem ser tanto positivos como negativos.

Quando o número inteiro é muito grande nós podemos usar a vírgula , para torná-lo mais legível, por exemplo 258,695,789.

Em Perl, a vírgula , é substituída pelo sublinhado _.

Sendo assim, o número 258,695,789 se torna 258_695_789.

Exemplo:

my $big_num = 258_695_789;

print("$big_num\n");

Saída:

258695789

O Perl não usa nem a vírgula , nem o sublinhado _ para manipular o número. O uso do sublinhado _ no código é somente para tornar o número mais legível.

Número de ponto flutuante em Perl

Um número de ponto flutuante em Perl contém uma parte inteira e uma parte fracionária.

Esse número possui a parte inteira e fracionária separadas por um ponto decimal. Números de ponto flutuante em Perl podem ser positivos ou negativos.

A parte fracionária do número pode ser somente positiva.

Os números de ponto flutuante podem ser representados por duas notações, ponto fixo e científico.

Notação de ponto fixo

Um número de ponto fixo em Perl possui a parte decimal fixa.

A parte decimal do número representa o início da fração.

Exemplo:

20.589

Notação científica

A notação científica em Perl é usada para representar um número muito grande ou um número muito pequeno.

A notação científica em Perl tem a seguinte sintaxe:

men

Onde m representa a mantissa, e é a notação científica para o Perl e o n é a ordem de grandeza ou expoente.

Veja também:
Notação científica (Wikipedia) 

Exemplo:

#!/usr/bin/perl
use strict;
use warnings;

my $x = 7.58325e2; # 758.325
my $y = 4e-4; # 0.0004

print("$x\n");
print("$y\n");

Saída:

758.325
0.0004

Apesar de declararmos os números em notação científica, o Perl retornou os números em notação de ponto fixo.

Também é possível indicar para o Perl que a representação dos números deve ser em notação científica.

Para isso, use a função printf com o placeholder %e.

Exemplo:

#!/usr/bin/perl
use strict;
use warnings;

my $x = 7.58325e2; # 758.325
my $y = 4e-4; # 0.0004

printf("Valor de X em notação científica: %e\n",  $x);
printf("Valor de Y em notação científica: %e\n",  $y);

Saída:

Valor de X em notação científica: 7.583250e+02
Valor de Y em notação científica: 4.000000e-04

Número hexadecimal em Perl

O Perl permite que o usuário represente um número em formato hexadecimal.

O sistema hexadecimal representa um número em base 16.

O sistema hexadecimal possui 16 símbolos de 0 à 9 e de a à f, onde a é equivalente a 10 e f a 15.

Em Perl, um número hexadecimal é precedido por 0x e pode ser representado de forma positiva ou negativa.

Exemplo:

$x = 0xa; # 10
$y = 0x3; # 3
$z = -0xe; # -14

print("$x\n");
print("$y\n");
print("$z\n");

Saída:

10
3
-14

Para representar o número em formato hexadecimal, use a função printf com o placeholder %x.

Exemplo:

$x = 0xa; # 10
$y = 0x3; # 3

printf("Valor de X em hexadecimal: %x\n", $x);
printf("Valor de Y em hexadecimal: %x\n", $y);

Saída:

Valor de X em hexadecimal: a
Valor de Y em hexadecimal: 3

Sistema de numeração hexadecimal (Wikipedia) 

Número octal em Perl

Para representar números em notação octal, preceda o número com 0 seguido do valor em base 8.

Em Perl, os números octais podem ser positivos ou negativos.

Exemplo:

$x = 032; # 26
$y = -0467; # -311

print("$x\n");
print("$y\n");

Saída:

26
-311

Para que o formato de saída dos números seja em octal, use a função printf com o placeholder %o.

Exemplo:

$x = 032; # 26
$y = 0171; # 121

printf("Valor de X em octal: %o\n", $x);
printf("Valor de Y em octal: %o\n", $y);

Saída:

Valor de X em octal: 32
Valor de Y em octal: 171

Veja também:
Sistema octal (Wikipedia) 

Número binário em Perl

O sistema binário representa números em base 2. O sistema possui somente dois valores, 0 e 1.

Para criar um número binário em Perl, coloque 0b antes do valor binário a ser representado.

Os números podem ser positivos ou negativos.

Exemplo:

$x = 0b01110101; # 117
$y = -0b110100; # -52

print("$x\n");
print("$y\n");

Saída:

117
-52

Para converter a saída do número em binário, use a função printf e o placeholder %b.

Exemplo:

$x = 0b01110101; # 117

printf("Valor de X em octal: %b\n", $x);

Saída:

Valor de X em binário: 1110101

Sistema de numeração binário (Wikipedia) 

Gerando um número aleatório em Perl

O Perl possui a função interna rand para gerar número aleatórios.

A função pode receber um número positivo ou negativo.

O retorno é um número aleatório de ponto flutuante entre 0 e o número passado como argumento para a função.

Exemplo:

O código abaixo retorna um número aleatório de ponto flutuante entre 0 e 10.

#!/usr/bin/perl
use strict;
use warnings;

my $random = rand(10);

print("Número aleatório entre 0 e 10: $random\n");

Saída:

Número aleatório entre 0 e 10: 2.178205233804

É possível converter o número aleatório de ponto flutuante para um número inteiro.

Para isso, use a função interna do Perl int.

A função int do Perl retorna apenas a parte inteira do número. A função int não executa nenhum tipo de arredondamento do número.

Exemplo:

#!/usr/bin/perl
use strict;
use warnings;

my $random_int = rand(10);

$random_int = int($random_int);

print("Número aleatório entre 0 e 10: $random_int\n");

Saída:

Número aleatório entre 0 e 10: 6
