Perl - Operadores

Os operadores Perl são símbolos que representam uma operação específica a ser executada em um ou mais operandos. Os operadores são usados ​​em expressões, que são instruções que usam valores e operadores para produzir um valor.

Os operadores são uma parte extremamente importante da programação.

Eles dizem ao seu computador o que você deseja que ele faça com uma determinada entrada.

Operadores numéricos

Perl tem uma variedade de operadores numéricos para ajudá-lo a manipular números, incluindo operações aritméticas, booleanas e bit a bit.

Operadores aritméticos

Perl tem muitos operadores matemáticos embutidos que são usados ​​para aritmética básica, como adição +, subtração -, multiplicação * e divisão /.

Esses operadores podem ser usados ​​com números ou sequências de texto para executar tarefas específicas.

Exemplo:

Adição e subtração de valores.

#!/usr/bin/perl
use strict;
use warnings;

print(10 + 10, "\n");
print(120 - 30, "\n");
print(1280 + 3567, "\n");
print(123 - 2866, "\n");

Saída:

20
90
4847
-2743

Quando você divide ou multiplica números em Perl, você usa o símbolo de divisão / e o símbolo de multiplicação * .

A ordem em que eles são inseridos não importa.

Exemplo:

Multiplicando e dividindo valores.

#!/usr/bin/perl
use strict;
use warnings;

print(10 * 10, "\n");
print(120 / 10, "\n");
print(1280 * 1.234, "\n");
print(123 / 2866, "\n");

Saída:

100
12
1579.52
0.0429169574319609

Os operadores de multiplicação e divisão têm maior precedência que os operadores de adição ou subtração em Perl, portanto, ao lidar com esses dois tipos de operações, ele realizará os cálculos em ordem.

Para entender o que o interpretador fará ao executar uma operação matemática, você deve saber a ordem em que ele executará as operações. Isso é conhecido como precedência do operador.

Na computação, a precedência do operador é um conjunto de regras que determina quais operações devem ser executadas primeiro quando várias operações são executadas ao mesmo tempo.

Exemplo:

#!/usr/bin/perl
use strict;
use warnings;

print(10 * 10 - 2 / 3 + 14, "\n");

Saída:

113.333333333333

Você pode usar parênteses para forçar o Perl a realizar o cálculo desejado primeiro.

Exemplo:

#!/usr/bin/perl
use strict;
use warnings;

print(10 * (10 - 2) / (3 + 14), "\n");

Saída:

4.70588235294118

Elevando um número à potência de outro número em Perl

Quando você deseja elevar um número à potência de outro número, você usa o operador de exponenciação **.

Exemplo:

Neste exemplo, queremos elevar 2 à potência de 4.

#!/usr/bin/perl
use strict;
use warnings;

print(2 ** 4, "\n"); # 2 * 2 * 2 * 2 = 16

Saída:

16

Obtendo o restante da divisão em Perl

Um operador matemático comum que usamos todos os dias é o operador módulo.

O operador de módulo % é usado para obter o resto da divisão de um número por outro.

Exemplo:

#!/usr/bin/perl
use strict;
use warnings;

print(13 % 4, "\n");
print(20 % 7, "\n");
print(100 % 13, "\n");

Saída:

1
6
9

O operador módulo pode ser usado para descobrir se um número é par ou ímpar.

O resto de um inteiro dividido por dois é 0 se o inteiro for par e 1 se o inteiro for ímpar.

Exemplo:

#!/usr/bin/perl
use strict;
use warnings;

print(10 % 2, "\n"); # 0 - Even
print(13 % 2, "\n"); # 1 - Odd

Saída:

0
1

Comparando valores numéricos em Perl

A linguagem de programação Perl fornece todos os operadores de comparação para números.

Operador Descrição Exemplo Resultado
== Igual 10 == 10 verdadeiro
!= Não igual 10 != 10 falso
< Menor que 10 < 130 verdadeiro
> Maior que 10 > 130 falso
<= Menor ou igual 10 <= 10 verdadeiro
>= Maior ou igual 10 >= 2 verdadeiro

Existe mais um operador em Perl usado para comparações numéricas, o operador de nave espacial <=>.

Também conhecido como operador de comparação combinada, retorna 0 se os números forem iguais, 1 se o primeiro for maior e -1 se o segundo for maior.

Exemplo:

#!/usr/bin/perl
use strict;
use warnings;

my $x = 10;
my $y = 20;

print($x <=> $y, "\n");

$y = 10;
print($x <=> $y, "\n");

$y = 5;
print($x <=> $y, "\n");

Saída:

-1
0
1

Operadores de string em Perl

Operadores de string em Perl são usados ​​para manipular e examinar strings de texto.

Operadores de comparação de strings

Perl tem muitos operadores de comparação de strings para comparar uma string com outra.

Operador Descrição Exemplo Resultado
eq Igual “abc” eq “xyz” falso
ne Não igual “abc” ne “xyz” verdadeiro
lt Menor que “abc” lt “xyz” verdadeiro
gt Maior que “abc” gt “xyz” falso
le Menor ou igual “abc” e “xyz” verdadeiro
ge Maior ou igual “abc” ge “xyz” falso
cmp Comparação “abc” cmp “xyz” -1

Operadores de concatenação de strings

Os operadores de concatenação de strings em Perl permitem combinar strings de texto em uma única string.

Exemplo:

#!/usr/bin/perl
use strict;
use warnings;

my $str1 = "Good";
my $str2 = "Morning.";

print($str1 . " " .  $str2, "\n");

Saída:

Good Morning.

O operador “x” em Perl é um operador de repetição. Ele repete o token ou grupo de tokens determinado x número de vezes.

Exemplo:

#!/usr/bin/perl
use strict;
use warnings;

my $str1 = "Perl";

print($str1 x 4, "\n");

Saída:

PerlPerlPerlPerl

Operadores lógicos

Operadores lógicos são usados ​​para avaliar declarações para produzir um valor verdadeiro ou falso.

A tabela a seguir lista os operadores lógicos que o Perl fornece:

Operador Nome Descrição
&& AND Avalia seu operando esquerdo e se for verdadeiro, avalia seu operando direito e retorna o valor do operando direito. Se o operando esquerdo for falso, ele não avaliará nem retornará nada.
|| OR Avalia seu operando esquerdo e se for falso, avalia seu operando direito e retorna o valor do operando direito. Se o operando esquerdo for verdadeiro, ele retornará o valor do operando esquerdo.
! NOT Nega uma declaração que o segue. Por exemplo: !(3 > 2) retornaria false porque 3 é maior que dois, o que retorna true, portanto, ao usar o operador NOT ! este valor é invertido para falso.

Exemplo:

#!/usr/bin/perl
use warnings;
use strict;

print(1 && 0, "\n"); # false or 0
print(1 || 0, "\n"); # true or 1
print( !(1 == 1) , "\n"); # false or empty

Saída:

0
1
