Perl - Strings

Uma string em Perl é um conjunto de caracteres entre aspas simples '' ou dupla "".

Há diferença entre criar uma string usando aspas simples '' e aspas dupla "". Mais sobre isso ao decorrer do tutorial.

Exemplo:

"Olá, como vai você?"
"Hoje é um belo dia."
'Perl é uma linguagem de programação bem fácil de aprender'

Vale lembrar que, o que realmente define uma string em Perl não são as aspas ao redor do conteúdo e sim a operação executada no valor. O Perl trata tanto número quanto string como tipo de dado scalar, o que diferencia um do outro é o comando executado sobre eles.

Manipulando strings em Perl

O Perl permite executar três tipos de operações em strings:

  • Operadores de alteração,
  • Operadores de comparação,
  • Funções de string predefinidas.

Operadores de alteração de strings em Perl

O Perl possui dois tipos de operadores de alteração em strings:

  • Concatenação: representado pelo operador .,
  • Repetição: representado pelo operador x.

Concatenando strings em Perl

Para concatenar uma string em Perl, use o operador ..

Exemplo:

$alpha1 = "abc";
$alpha2 = "def";

print($alpha1 . $alpha2);

abcdef

Repetindo strings em Perl

Para repetir uma string em Perl, use o operador x, seguido pelo número de vezes que a string deve ser repetida.

Exemplo 1:

print("Esse texto contém um sublinhado embaixo.\n", "-"x40);

Saída:

Esse texto contém um sublinhado embaixo.
----------------------------------------

Exemplo 2:

print("Perl" x10);

PerlPerlPerlPerlPerlPerlPerlPerlPerlPerl

Funções de string predefinidas

O Perl possui diversas funções internas para manipular strings.

Comprimento da string em Perl

O Perl possui um função interna usada para retornar o comprimento da string.

Para saber a quantidade de caracteres que uma string possui, use a função length.

Exemplo:

$str1 = "Perl é simples de entender.";

print( length($str1), "\n");

Saída:

28

Mudando a caixa das letras em Perl

Em Perl, é possível converter letras maiúsculas para minúsculas e vice-versa.

Use a função uc para converter a string para maiúsculas.

Exemplo:

$str1 = "Perl é simples de entender.";

print( uc($str1), "\n");

Saída:

PERL é SIMPLES DE ENTENDER.

Para converter uma string para minúsculas, use a função lc.

Exemplo:

$str1 = "PERL é SIMPLES DE ENTENDER.";

print( lc($str1), "\n");

Saída:

perl é simples de entender.

uc significa uppercase, traduzindo para o português seria maiúscula e lc significa lower case, ou seja, minúscula em português.

Encontrado uma substring dentro de uma string em Perl

O Perl possui duas funções internas usadas para encontrar uma substring dentro de uma string, index e rindex.

  • A função index procura uma substring a partir da posição indicada nos argumentos da função. Ela retorna o número da posição da primeira ocorrência da substring procurada. Se o parâmetro que indica por onde iniciar a procura for omitido, a função começa a procurar a partir do início da string.

  • A função rindex funciona da mesma forma, só que ela inicia a procura da substring a partir do final da string.

Exemplo:

$str1 = "Eu estou aprendendo a programar em Perl, é uma linguagem muito poderosa e simples de aprender.";

$subStr = "Perl";

$position = index( $str1, $subStr );

print( $position, "\n");

Saída:

35

No exemplo anterior, a função index encontrou a primeira ocorrência da substring Perl na posição 35 da string armazenada na variável $str1.

Extraindo uma substring de uma string em Perl

A função interna substr do Perl é usada para extrair ou modificar substrings dentro de strings.

A função substr recebe como argumentos:

  • A string, de onde a substring será extraída,
  • Um número representado de onde a pesquisa pela substring deve partir.

Para iniciar a pesquisa a partir do final da string use um número negativo.

  • Outro número indicando a quantidade de caracteres da substring que deve ser extraída.

Exemplo 1:

O exemplo abaixo extrai duas substrings da string principal $str.

$str = "Minha fruta favorita é o morango.";

$food = substr($str, 6, 5); # fruta
$fruit = substr($str, -8, 7); # morango

print("$food:$fruit\n");

Saída:

fruta:morango

Exemplo 2:

O exemplo abaixo substitui a substring morango por abacaxi na string principal $str.

$str = "Minha fruta favorita é o morango.";

substr($str, -8, 7, "abacaxi"); # morango

print("$str\n");

Saída:

Minha fruta favorita é o abacaxi.

Delimitadores alternativos — Para criar strings em Perl

Em Perl, é possível usar delimitadores alternativos as aspas para criar strings.

Exemplo:

$str1 = q/Perl é simples de entender./;
$str2 = qq/Perl é uma linguagem de programação poderosa./;

print($str1, "\n");
print($str2, "\n");

Saída:

Perl é simples de entender.
Perl é uma linguagem de programação poderosa.

No exemplo anterior:

  • A sintaxe q/string/ é equivalente a 'string',
  • A sintaxe qq/string/ é equivalente a "string".

O operador q é chamado de quote-like, e pode ser usado para substituir as aspas.

O uso da barra / no lugar das aspas também pode ser substituído por qualquer caractere não alfanumérico, ou seja, não podem ser letras nem números.

Exemplo:

O código abaixo substitui a barra / por ~.

$str1 = q~Perl é simples de entender.~;
$str2 = qq~Perl é uma linguagem de programação poderosa.~;

print($str1, "\n");
print($str2, "\n");

Saída:

Perl é simples de entender.
Perl é uma linguagem de programação poderosa.
