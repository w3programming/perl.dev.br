Perl - Sintaxe

Perl possui uma sintaxe muito simples e direta.

Vários elementos da linguagem são semelhantes ao idioma inglês.

Variáveis e valores

Em Perl, nós criamos programas para manipular algum tipo de dado.

O Perl possui diversos tipos diferentes de dados.

Esses tipos de dados incluem strings, números e dados mais complexos como listas.

Exemplo:

"Programando em Perl"
120
1.2

Um dos conceitos mais importantes em programação é a reusabilidade.

Para reusar valores nós precisamos armazená-los em algum lugar.

Em Perl, nós usamos variáveis para armazenar valores, reusá-los e processá-los.

Exemplo:

$name = "Peter";
$x = 10;
$z = 10.567;

No exemplo anterior:

  • A variável $name possui um valor do tipo string,
  • A variável $x possui um valor do tipo número inteiro,
  • E a variável $z possui um valor do tipo número de ponto flutuante.

Expressões em Perl

Uma expressão em Perl, é qualquer coisa que retorne um valor.

Uma expressão em Perl pode ser desde um número literal até uma chamada de função.

Exemplo:

10

O número literal 10 é considerado uma expressão em Perl que retorna o mesmo valor, 10.

Exemplo:

$num1 + $num2

$num1 + $num2 é uma expressão em Perl que retorna a soma dos valores das variáveis $num1 e $num2.

Uma expressão em Perl pode ser usada em um outra expressão maior ou em uma instrução.

Instruções em Perl

Uma instrução em Perl é composta por expressões.

Uma instrução em Perl deve ser finalizada por um ponto e vírgula ;.

Exemplo:

$num1 = 10;
$num2 = 20;

$total = $num1 + $num2;

print($total);

Saída:

30

Cada linha do código de exemplo representa uma instrução em Perl.

Comentários em Perl

Para criar um comentário em Perl, inicie a linha com uma hash #

Exemplo:

# Essa linha é um comentário em Perl.
print("Comentários em Perl.\n")

Saída:

Comentários em Perl.

Comentários em Perl podem ser usados para documentar o código.

Exemplo:

# Define uma variável com o valor do salários do funcionário.
$salary = 1220;

# Imprimi o salário atual.
print("Salário atual: " . $salary . "\n");

# Aumenta em 10% o salário atual.
$salary = $salary * 1.10;

# Imprimi o salário com o aumento de 10%.
print("Salário atual com aumento: " . $salary . "\n");

Saída:

Salário atual: 1220
Salário atual com aumento: 1342

Também é possível usar comentários na mesma linha.

Exemplo:

print("Olá, tudo bem?\n"); # Imprimi uma frase na tela.

Saída:

Olá, tudo bem?

Blocos

Um bloco em Perl é composto por uma ou mais instruções entre parenteses {}.

Exemplo:

{
    $num1 = 10;
    $num2 = 20;

    $total = $num1 + $num2;

    print($total);
}

Saída:

30

Blocos em Perl podem ser usados para separar pedaços de códigos reutilizaveis.

Blocos são importantes para organizar o programa e torná-lo mais simples de entender.

Variáveis declaradas dentro de um bloco possuem escopo limitado. Somente instruções e expressões de dentro do próprio bloco terão acesso a essas variáveis.

Espaços em branco

Espaços em branco em Perl são compostos por:

  • espaços,
  • tabulações,
  • novas linhas,
  • ou uma combinação desses elementos.

Espaços em branco em Perl são usados para deixar o código mais legível.

O interpretador do Perl ignora os espaços em branco.

Exemplo:

$num1 = 10;
$num2=20;
$num3             =30;

O script acima é válido para o Perl e os espaços em branco não fazem nenhuma diferença para o interpretador.

Sempre use espaços em branco para tornar o seu código mais legível e fácil de compreender.

Palavras-chave

Em programação, uma palavra-chave é um termo reservado pela linguagem.

Uma palavra-chave pode significar uma função interna, um comando, um pragma ou possuir outro significado especial.

Palvras-chave da linguagem de programação, normalmente, não podem ser reutilizadas pelo usuário.

Por exemplo, o Perl possui a palavra-chave print reservada para a função print().

Não seria uma boa escolha nomear uma variável como $print ou criar uma função usando o mesmo nome.

Isso pode gerar confusão se algo de errado ocorrer no seu código.

Você pode também sobreescrever alguma função importante interna da linguagem.

Vale ressaltar que muitas linguagens de programação não permitem que os usuários sobreescrevam palavras reservadas, justamente para evitar confusão com os termos internos.
