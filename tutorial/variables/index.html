Perl - Variáveis

Variáveis em Perl são usadas para armazenar dados.

Uma variável é um recursos excencial para qualquer linguagem de programação.

O Perl fonece três tipos de variáveis:

  • Scalars,
  • Listas,
  • e Hashes.

Nesse tutorial nós vamos aprender sobre o tipo Scalar.

Criando uma variável

Uma variável em Perl é criada usando o cifrão $ seguido pelo identificador da variável e um valor de atribuição.

Assim que um valor é atribuido para a variável, ela é criada pelo Perl.

Exemplo:

$name = "Richard";
$age = 26;

print("Nome: " . $name . "\n");
print("Idade: " . $age . "\n");

Saída:

Nome: Richard
Idade: 26

Variáveis em Perl não possuem um tipo fixo.

É possível alterar o tipo de dado da variável em tempo real.

# $var contém um dado do tipo String.
$var = "Richard";
print($var);

# Agora $var contém um dado do tipo número inteiro.
$var = 10;
print($var);

Saída:

Richard
10

Variáveis Scalar em Perl

Uma variável Scalar em Perl é usada para manipular valores do tipo número e string.

Exemplo:

$name = "Jorge";

Para criar uma variável Scalar válida em Perl é necessário:

  • Inserir um cifrão $ no ínicio,
  • Seguido por uma letra ou um sublinhado,
  • Seguido por qualquer combinação de letras, números ou sublinhados,
  • O comprimento máximo para o nome da variável é de 255 caracteres.

O exemplo a seguir contém nomes válidos de variáveis em Perl.

Exemplo:

$name = "Richard";
$full_name = "Richard Nonato";
$_10 = 10;
$__port__to__use__ = 80;

print("Nome: $name\n");
print("Nome completo: $full_name\n");
print("Valor: $_10\n");
print("Porta: $__port__to__use__\n");

Saída:

Nome: Richard
Nome completo: Richard Bela
Valor: 10
Porta: 80

O código abaixo apresenta nomes inválidos de variáveis em Perl.

$full-name = "Richard Bela";
$10 = 10;
$port to use = 80;

print("Nome completo: $full-name\n");
print("Valor: $10\n");
print("Porta: $port to use__\n");

Os identificadores de variáveis do exemplo anterior são considerados inválidos pelas seguintes razões:

  • $full-name possui um sinal de subtração - no meio do identificador,
  • $10 inicia o nome da variável com um número, o que não é permitido,
  • $port to use inclui espaços em branco no nome da variável, o que também gera um erro.

Como dever de casa, tente consertar o código anterior e alterar os identificadores das variáveis para nomes válidos em Perl.

Criando variáveis com o comando “my” em Perl

O Perl permite criar variável conforme a necessidade do programador.

Uma variável em Perl não precisa ser declarada explicitamente.

Porém, essa liberdade pode gerar erros de programação.

Exemplo:

$name = "Richard";

print("Meu nome é $nayme\n");

Saída:

Meu nome é

No exemplo anterior, o identificador nayme usado como argumento para a função print está errado.

Mesmo assim o interpretador do Perl tentou acessar o valor contido na variável nayme.

A variável nayme não existe, mas mesmo assim o interpretador do Perl executou o script.

Para evitar situações como essa basta indicar para o interpretador do Perl que as variáveis devem ser declaradas explicitamente.

Usando o pragma use strict logo no ínicio do script, o interpretador do Perl checa se as variáveis usadas foram declaradas e existem no contexto do programa.

Exemplo:

#!/usr/bin/perl
use strict;

my $name = "Richard";

print("Meu nome é $nayme\n");

Saída:

Global symbol "$nayme" requires explicit package name (did you forget to declare "my $nayme"?) at script.pl line 6.

Execution of script.pl aborted due to compilation errors.

Agora o interpretador gera um erro e indica qual foi a variável que interferiu na execução do script.

Após iniciar o seu script com o pragma use strict é necessário criar as variáveis usando o comando my.

Ao usar o comando my não é necessário atribuir um valor imediatamente para a variável.

Exemplo:

#!/usr/bin/perl
use strict;

my $name;

$name = "Richard";

print("Meu nome é $name\n");

Saída:

Meu nome é Richard

Escopo de variáveis

Em Perl, as variáveis possuem escopo de bloco.

O escopo de variável se refere a limitação do código em poder acessar uma variável dependendo do bloco que ela foi criada.

Uma variável criada dentro de um bloco em Perl é considerada uma variável local. E só pode ser acessada dentro desse bloco.

Exemplo:

#!/usr/bin/perl
use warnings;

$fruit = "abacaxi";

print("Minha fruta favorita é o " . $fruit . "\n");

{
    my $fruit = "morango";
    print("Eu também gosto de " . $fruit . "\n");
}

print("Só para confirmar, minha fruta favorita é o " . $fruit . "\n");

Saída:

Minha fruta favorita é o abacaxi
Eu também gosto de morango
Só para confirmar, minha fruta favorita é o abacaxi

Vamos enteder melhor o exemplo anterior:

  • Logo no inicio do código, nós declaramos uma variável $fruit com o valor abacaxi.
  • Em seguida esse valor é apresentado na tela usando a função print.

Perceba que o valor impresso na tela é abacaxi.

  • Na sequência, um bloco é criado e dentro dele, criamos outra variável $fruit usando o comando my e atribuimos o valor morango
  • Ainda dentro do bloco o valor dessa variável é apresentado na tela, ou seja, morango.
  • E por fim, fora do bloco, colocamos o valor da variável $fruit na tela.

Novamente o valor apresentado é abacaxi, pois a variável $fruit de dentro do bloco não pode ser acessada fora dele, devido ao seu escopo local. Essa variável $fruit que acessamos ao final do código, é a variável $fruit global, declarada fora do bloco, logo no inicio do script.

Variável global em Perl

O Perl permite que o programador declare uma variável global de forma explicita.

Uma variável global é visível por todo o código e pode, tabém, ser acessada por pacotes de código externos.

Para criar uma variável global em Perl, use o comando our na declaração da variável.

Exemplo:

our $fruit = "abacaxi";

Interpolação de variáveis em Perl

Em Perl, é possível inserir variáveis diretamente dentro de strings.

A interpolação faz a junção do valor da string e do valor da variável sem a necessidade do uso do . para concatenação.

Exemplo:

#!/usr/bin/perl
use strict;
use warnings;

my $age = 22;
my $text = "Eu tenho $age anos de idade.\n";

print($text);

Saída:

Eu tenho 22 anos de idade.

Em Perl, só é possível interpolar variáveis scalar e vetores (arrays). O Perl não faz interpolação de variáveis quando essas são do tipo hash.

A interpolação em Perl somente funciona em strings usando aspas dupla "". Não é possível interpolar variáveis em strings com aspas simples ''.

Existe uma outra forma de concatenar strings que você já viu no decorrer do tutorial.

Usando o . também é possível juntar valores em Perl.

Exemplo:

my $text = "Eu tenho " . $age . " de idade.\n";

Nesse caso a variável fica fora das aspas.

A concatenação usando o ponto . também funciona com strings usando aspas simples ''.

Exemplo:

my $text = 'Eu tenho ' . $age . ' de idade.' . "\n";

O caracter de nova linha \n só funciona dentro de aspas dupla "". Se colocá-lo dentro de aspas simples '', ele será apresentado na tela como texto.
