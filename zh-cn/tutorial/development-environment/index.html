设置 Perl 开发环境

在本教程中，我们将安装 Perl 以及开始使用这种语言编程所需的一切。

要在 Perl 中编程，必须安装编程语言分发应用程序和一个简单的文本编辑器。

Strawberry Perl — 它是什么以及如何下载它

Strawberry Perl 是 Microsoft Windows 的 Perl 开发环境。

Strawberry Perl 包含运行 Perl 脚本所需的所有要求。

Strawberry Perl 可以直接从官网下载https://strawberryperl.com  。

下载后，只需运行安装程序应用程序。

在 Windows 上安装 Strawberry Perl — 使用图像一步一步

  1. 运行 Strawberry Perl 安装程序后，立即出现以下窗口。单击下一步按钮继续。

  1. 单击“我接受许可协议中的条款”复选框，然后单击“下一步”按钮，阅读并接受计划条款

  1. 选择应用程序安装位置。默认情况下，建议将 Strawberry Perl 安装在 C:\Strawberry 文件夹中。如果不确定安装位置，请保留此默认文件夹。

  1. 配置好整个系统后，只需单击安装按钮即可开始软件安装。

  1. 现在，只需等待安装完成。

  1. 完成 Strawberry Perl 安装后，将出现以下屏幕。单击“完成”按钮完成安装程序。

Perl 安装确认

要确认 Perl 已正确安装在您的计算机上，请执行以下步骤：

  1. 打开 Windows 命令提示符并键入命令 perl -v。
  2. 按 Enter 键执行命令。
  3. 检查该命令是否输出计算机上安装的 Perl 版本，如下图所示。

如果命令返回如下内容：

'perl' 不是内部或外部命令、可运行程序或批处理文件。

或者

'perl' 未被识别为内部命令...

…有必要验证目录 C:\Strawberry\perl\bin 是否已添加到 Windows PATH 环境变量中。

您可以使用以下命令在 PATH 环境变量中添加 Perl 目录：

setx path "%path%;C:\Strawberry\perl\bin"

安装文本编辑器 — 也称为代码编辑器

Perl 脚本是纯文本。

Perl 脚本可以在任何纯文本编辑器中进行编辑。

为了使 Perl 编程更容易，建议安装一个具有语法高亮和自动完成等功能的编辑器。

编辑器选项很多。我建议使用 Visual Studio Code。

要下载它，只需访问官方网站https://code.visualstudio.com  。

安装 Visual Studio Code 非常简单明了。运行安装程序并按照程序提供的步骤进行操作。

如果您想尝试其他编辑器选项，可以尝试Sublime Text  或Atom  。 程序员中最受欢迎的两种代码编辑器。 Notepad++  也是一个不错的选择。
