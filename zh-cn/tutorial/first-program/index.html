用 Perl 开发第一个程序

Perl 脚本是纯文本。

尽管它很简单，但 Perl 脚本可以执行复杂的任务。

  • 首先，在 C: 根目录中创建一个名为 PERL-SCRIPTS 的文件夹。
  • 打开您喜欢的文本编辑器以创建第一个 Perl 脚本。
  • 创建一个新文件并输入以下代码：
#!/usr/bin/perl
use warnings;
print("你好世界！\n");
  • 将名为 hello.pl 的文件保存到 perl-scripts 文件夹。

在 Perl 中运行脚本

要运行 Perl 脚本，请在 Windows 中打开命令提示符。

导航到脚本所在的文件夹。

在我们的例子中，文件夹位于 c:\perl-scripts。要将提示放在此文件夹中，请键入命令 cd c:\perl-scripts 并按 Enter 键。

运行命令：

perl hello.pl

输出：

你好世界！

如果您在屏幕上看到此结果，那么恭喜您刚刚创建并成功运行了您的第一个 Perl 脚本。

解释脚本 - 详细

让我们通过第一个 Perl 程序的每一行来了解它是如何工作的。

#!/usr/bin/perl

第一行告诉 shell 这个脚本应该使用 Perl 解释器运行。

地址 /usr/bin/perl 正是解释器所在的位置。

use warnings;

脚本的第二行包含一个 Pragma。

这个 Pragma 告诉 Perl 解释器，当代码中发生某些事情时，应该向用户显示额外的警告。

Perl 中使用的另一个非常重要的 Pragma 是 use stricts。随着教程的进展，我们将看到更多关于它的信息。

print("你好世界！\n");

脚本的第三行包含一个名为 print 的函数。

此函数在屏幕上放置一个值。

在这种情况下，值是括号内的任何值。

"你好世界！\n" 表示 Perl 中的字符串。

Perl 中的字符串用引号""括起来。

在整个教程中，您将了解有关 Perl 中字符串的更多信息。

字符串中的序列 \n 代表 Perl 解释器的换行符。

\n 被称为 escape sequence。
