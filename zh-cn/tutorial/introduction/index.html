Perl 简介

在本文中，您将了解 Perl 编程语言的一些重要方面。这是与构成该语言的特征的第一次接触。

Perl 特性——语言的重要方面

Perl 是市场上最流行、功能最丰富的编程语言之一。它经过二十多年的发展和改进。它在其发展过程中拥有一个积极参与的社区。

Perl 可以在大量的平台上运行。兼容大多数流行的操作系统，如 Windows、Linux 和 MacOS，对于需要跨平台可移植性的任何人来说，它都是明智的选择。

Perl 目前支持超过 100 种不同的平台。

但是 Perl 可以用来做什么呢？

Perl 是一种通用语言。它既可以用于任务关键型项目，也可以用于小型项目，甚至可以用于非常简单的单行脚本。

Perl 最重要的特性——语言亮点

1. 可扩展的编程语言

Perl 有一个名为CPAN  的模块存储库 ， Comprehensive Perl Archive Network的缩写，在其中您可以找到该语言的数十万个扩展。

如果您需要一个执行特定任务的模块，很可能社区中的某个人已经开发了它。

模块范围从图形用户界面 UI 到与最流行的数据库管理器的集成。

Perl 为MySQL、Oracle、PostgresSQL等程序员提供最流行的数据库支持。

2. 易于使用和学习

尽管 Perl 存在一些争议，但该语言非常容易学习。

除了拥有庞大而乐于助人的社区  ，您仍然可以在互联网上免费找到数千个教程。

Perl 也有文档  非常详细且易于理解。该语言以其在执行复杂任务方面的效率而著称。

3. 多功能性

任何编程语言最重要的特性之一是它在编程范式方面的多功能性。

Perl 支持面向对象、过程式编程或函数式编程，所有这些编程范例。

这允许程序员使用最适合他们项目的范例自由地开发他们的程序。

4.正则表达式

当我们谈论文本操作时，Perl 处于另一个层次。可以说，语言是正则表达式的先驱。

正则表达式或正则表达式是一个强大的系统，用于在各种类型的内容中查找文本出现。随着教程的进行，您将了解更多有关此系统的信息。

Perl 有一组专门用于文本处理的内置 API。有了它们，操作由 XML 和 HTML 等标记语言生成的文件要容易得多。

Perl 的正则表达式引擎不需要注释。凭借强大的系统，Perl 的正则表达式可用于通过复杂的组合查找系统上的特定文件。

5. 开源

开源项目允许用户自由、自由地访问、更改甚至分发它。

Perl 从一开始就是作为一个开源项目开发的。您可以下载 Perl，对其进行更改，并使用 GNU、GPL 或艺术许可证之一自由分发它。

该语言仍然支持商业用途。

6.质量安全

当我们谈论编程时，语言安全是必不可少的。

Perl 由 Coverity 认证，没有安全漏洞。语言核心也具有低缺陷密度。

这使用户对使用 Perl 开发他们的程序更有信心。

不同的 Perl 版本——版本 5 x 版本 6

这里值得注意的是，Perl 有两个主要版本，这在用户中会产生混淆。

在 Perl 的第 5 版之前，该语言仍称为 Perl。然而，版本 6 与 Perl 有很大的不同，以至于它变成了另一种语言，称为Raku  。

Raku 可以被认为是 Perl 的完全面向对象的实现。

在我们的教程中，我们将使用 Perl 5.3 版。

谁发明了 Perl？

Perl 于 1987 年由程序员 Larry Wall 首次引入。

他是 NASA 的系统管理员，需要一种语言来更有效地处理报告。Perl 的第一个版本是在 UNIX 操作系统上创建和运行的。

二十多年来，Perl 一直由一个活跃且敬业的程序员社区开发和改进。
